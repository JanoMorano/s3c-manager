S3C Manager — Layout Proposal (merged)

Service Governance Workspace · 6 sekcí v sidebaru · Service 360 header · Relationship studio · Readiness gate s human language · C3 Board · Decision log knowledge base · Impact dual-view · Import 6-step wizard.

Univerzální pravidlo každé stránky: Signal → Explanation → Action → Detail → Audit.
Manažer chápe nahoře, admin opravuje dole. Každý objekt má ownera, stav a další akci. Každý graf má textový závěr.
