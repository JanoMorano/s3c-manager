# V1.2 Release Notes

S3C Manager v1.2 is the post-reduction handover build focused on help alignment, current-product clarity, and Docker readiness. It does not introduce another reduction pass; it prepares the current governance cockpit for packaging and acceptance.

## Highlights

- Aligned Czech and English in-app help with the active application surface.
- Removed active-help wording for retired user workflows such as internal service requests, notification inbox tables, C3 board kanban, risk-radar/advisor surfaces, renewal-calendar surfaces, and bulk-folder import.
- Kept legacy variant evidence documented only as historical offering evidence where it is still part of the data/API model.
- Bumped Docker/runtime-visible application version metadata to `1.2` and package metadata to `1.2.0`.
- Added a final readiness and removal-candidate report for handover review.
- Completed final sunset cleanup for retired API modules, legacy frontend redirect aliases, and `/help-2-*` compatibility manuals.
- Completed final commit-readiness refresh on 2026-05-07 with middleware lint cleanup, full middleware tests, security audits, Docker rebuild and smoke verification.
