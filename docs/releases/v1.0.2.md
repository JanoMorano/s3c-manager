# V1.0.2 Release Notes

## Title
ITIL-Ready Service Catalogue — Lifecycle, Offerings & Service Onboarding

## Summary
Introduces full ITIL catalogue capabilities: service lifecycle governance (`draft → live → retired`), service offerings, support model, requestability, consumer value, audience and eligibility, and operational links.

Adds a guided New Service Wizard with inline ITIL hints, modernises the UI with shadcn/ui components and Tailwind CSS, and upgrades the service detail with business-focused tabs (`Overview`, `Offerings`, `Request & Eligibility`, `Support`, `Governance`).

## Key Technical Notes

- App release version: `1.0.2`
- Default runtime schema version: `2.2.1`
- New additive schema files: `15_itil_catalogue_phase1.sql`, `16_consumer_value.sql`
- Existing service, auth, export, and stats routes remain backward compatible
