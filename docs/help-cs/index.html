Help rozcestník

S3C Manager - uživatelský a administrátorský manuál

Manuál je rozdělený do 12 samostatných kapitol. Každá kapitola vysvětluje, k čemu daná část aplikace slouží, co je na obrazovce vidět, jaké má uživatel možnosti, která pole co znamenají a jaké vazby stránka vytváří na služby, C3, governance, databázi, API nebo externí systémy.

Jak s manuálem pracovat

Nejrychlejší cesta

Pokud aplikaci vidíte poprvé, začněte podstatou aplikace a potom přejděte do kapitoly podle své role. Každá stránka manuálu má stejný rytmus: účel obrazovky, screenshot slot, pole a volby, vazby na jiné části aplikace, dopad na data a doporučený postup.

Pravidlo screenshotů

Screenshot sloty jsou připravené už teď. Stačí nahrát PNG do docs/help-cs/img pod názvem uvedeným v popisku. Text kapitoly zůstává ručně psaný, aby se po UI změně nepřepsala vysvětlení polí a vazeb.
