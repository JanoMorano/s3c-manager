Konfigurace systému

Nastavte základní parametry aplikace. Tato konfigurace bude uložena do databáze.

✓DATABASE_URLenv nastaven
✓JWT_SECRETsecret nastaven
✓Verze aplikacev1.0.2
🔒 Secrets (hesla, JWT klíče) se nikdy neukládají do databáze. Jsou čteny výhradně z prostředí při startu kontejneru.

Validace secrets

Všechny kritické secrets jsou ověřeny při startu aplikace a nikdy nejsou ukládány do databáze.

Secrets jsou čteny pouze z prostředí (Docker ENV, secret mount). Průvodce je zobrazuje jen pro ověření přítomnosti.
✓JWT_SECRET přítomenověřeno při startu
✓DATABASE_URL přítomnaověřeno při startu
✓Mimo zdrojový kódcompose env / secret mount
✓V lozích nejsou plain-text secretsžádné secrets v lozích
⚠ Hesla nikdy nezadávejte přímo do kódu nebo docker-compose.yml. Použijte .env soubor nebo Docker secrets.

Administrátorský účet

Vytvořte první administrátorský účet. Tento účet bude mít plná práva k systému.

🔐 Heslo musí mít min. 10 znaků, velké písmeno, malé písmeno, číslo a speciální znak.

Moduly

Vyberte, které moduly chcete aktivovat. Základní katalog je vždy povinný.

Service Catalogue Core Povinný
Základní katalog služeb — správa životního cyklu, offerings, support model, requestability a governance.
C3 Taxonomy
Rozšiřuje katalog o mapování na Business Capabilities, Aplikace, Datové objekty a Technology Interactions. Umožňuje vizualizaci závislostí.

Data packages

Zvolte, jaká data chcete nainstalovat. Povinné balíčky zajistí správné fungování systému.

🗄️ Core installation Povinný
Schéma databáze, migrace, platformové tabulky.
📋 Reference seed — Service Catalogue Povinný
Číselníky: typy služeb, SLA tiers, support hours, request channels, lifecycle states.
🧩 Reference seed — C3 Taxonomy Povinný pro C3
📦 Business data — Service Catalogue Volitelný
Vlastní CSV/XLSX import v kroku 8.
🧪 Demo data Doporučeno
Ukázkové služby pro rychlý start a testování UI.
Povinné balíčky se vždy nainstalují automaticky. Demo data jsou doporučena pro první instalaci.

Import souborů

Nahrajte vlastní data ve formátu CSV, XLSX nebo JSON. Tento krok je volitelný.

📤
Přetáhněte soubory sem nebo klikněte pro výběr
Podporované formáty: .csv, .xlsx, .json · Každý soubor = jedna dávka importu
Každý soubor tvoří samostatnou importní dávku. Chybné řádky v dávce nezastaví import celého souboru.

Náhled importu

Import byl přeskočen. Vlastní data lze importovat kdykoli po instalaci přes sekci Administration.

Žádné soubory nebyly nahrány. Systém nainstaluje pouze povinná a vybraná reference data.
